Introducción a los Layout CSS

Display Block

  • El contenido del elemento toma el 100% del width
  • Se puede usar margin y padding por todos los lados
  • La mayoría de las etiquetas vienen por Default con un display:block

Display inline

  • Cada caja ocupa un espacio
  • No puedo usar margin, ni padding arriba ni abajo, solo a la derecha e izquierda.
  • Tampoco se puede aplicar width o height.
  • Etiquetas como span viene por Default con un display:inline

  • Puedo definir un width y height
  • El contenido es del mismo tamaño

Display Grid

  • Definir columnas: grid-templates-columns
  • Definir filas: grid-templates-rows
  • Definir espaciados: gap
  • Justificación del contenido:
    justify-content start | center | end | stretch | space-around | space-evenly | space-between
    justify-items start | center | end | stretch
  • Alineación del contenido:
    align-content start | center | end | stretch | space-around | space-evenly | space-between align-items start | center | end | stretch
